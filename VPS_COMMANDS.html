3. Database (PostgreSQL)

🗄️ เข้า psql
คำสั่งทำอะไร
sudo -u postgres psql -d contractflowเข้า DB contractflow
\dtดู table ทั้งหมด
\qออกจาก psql
👤 จัดการ User
คำสั่งทำอะไร
SELECT id, email, name, role, active FROM "User";ดู user ทั้งหมด
node -e "const b=require('bcryptjs'); b.hash('NewPass123',10).then(h=>console.log(h))" generate bcrypt hash สำหรับ password ใหม่
UPDATE "User" SET "passwordHash"='HASH' WHERE email='user@example.com'; เปลี่ยน password ใน DB (แทน HASH ด้วย bcrypt hash)
UPDATE "User" SET active=false WHERE email='user@example.com'; ปิดการใช้งาน user
📋 สร้าง Admin คนแรก (กรณี reset DB)
ขั้นตอนคำสั่ง
1. Generate hash node -e "const b=require('bcryptjs'); b.hash('YourPassword',10).then(h=>console.log(h))"
2. Insert admin sudo -u postgres psql -d contractflow -c "INSERT INTO \"User\" (id, email, name, role, department, \"passwordHash\", active) VALUES (gen_random_uuid(), 'admin@innopower.co.th', 'Admin', 'ADMIN', 'IT', 'HASH_HERE', true);"
🔍 Query ที่ใช้บ่อย
คำสั่งทำอะไร
